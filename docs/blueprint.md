# **App Name**: GuineaBiz

## Core Features:

- Company Directory: Search registered companies by name, category, or services offered by a company. (include company contact info, description, products, services (list from services directory), working hours) etc
- Services and Procedure Guide: Browse a directory of government services and procedures with descriptions, requirements, and instructions.
- Institutions Details: Be able to see institutions, see the procedures that can be done there, contact information.
- Procedure steps: If you open a procedure you should she the steps to complete it, where, and what it costs.
- Procedure Details: Display step-by-step instructions for each government procedure, with institution, required info, and location.
- Interactive Map: Interactive map displaying the location of companies, branches, and government institutions.
- Favorites List: Allow users to save favorite companies and procedures for quick access.
- Report Info: Users can report inaccurate or outdated information on company profiles and service details. Incorporate this into LLM output using tool usage, to maintain data accuracy.
- Add Reviews: Customers/users can add reviews on companies, institutions, procedures, services.
- Review Summaries: Utilize a tool to provide generative AI summaries of user reviews of Equitorial Guinean businesses.
- Signin Page: Add signin page, to be able to list companies or institutions ( hence, add their services, products or procedures)
- Language: Everything should be in spanish

## Style Guidelines:

- Primary color: Vibrant green (#3CB371) to symbolize growth, prosperity, and Equatorial Guinea's lush environment.
- Background color: Light, desaturated green (#E0EEE8) to provide a calming and professional backdrop.
- Accent color: Gold (#D4A274), representing the country's rich resources.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern look.
- Use clean and simple icons to represent categories, services, and actions, in a style appropriate to wayfinding.
- Maintain a clean, grid-based layout to ensure information is easy to find and navigate.
